### 最终方案：PC生图服务器

#### 网络架构
```
Mac(公司) → EasyTier → iStoreOS(10.10.10.3) → PC(10.10.10.20)
                      (防火墙需开放端口)
```

#### PC配置要点
- **已安装**：ComfyUI（E:\ai\ComfyUI）
- **待安装**：Ollama（E:\ai\）
- **模型存储**：E:\ai\ollama_models
- **测试模型**：flux-schnell（速度）+ flux-dev（质量）
- **显存需求**：schnell ~8GB，dev ~10-12GB（4070 12GB可跑）

#### iStoreOS防火墙配置
需要开放端口：
- 11434（Ollama API）
- 8188（ComfyUI API）

#### 生图使用流程
1. 用户用中文描述想要什么画面
2. AI自动转成英文提示词
3. 通过EasyTier网络调用PC的Ollama/ComfyUI API
4. 返回生成的图片
